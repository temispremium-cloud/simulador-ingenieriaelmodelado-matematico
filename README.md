## 📝 Sobre el Proyecto

Este proyecto es un simulador avanzado diseñado para resolver y visualizar ecuaciones diferenciales. Es una herramienta técnica robusta pensada para estudiantes y profesionales de la Universidad de La Guajira.

> [!TIP]
> Puedes ver la versión desplegada en vivo aquí: [https://remitocle328.github.io/Uniguajira/](https://remitocle328.github.io/Uniguajira/)

**Requisitos previos:** Node.js
